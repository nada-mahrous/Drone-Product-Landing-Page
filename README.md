# Drone Product Landing Page

This project is a product landing page for the **DJI Mavic 4 Pro** drone.

The goal of the project was to build a clean and organized layout that highlights:
- Product specifications
- Customer reviews
- Related products

## 🛠 Technologies Used
- HTML
- CSS

## ✨ Features
- Clean product-focused layout
- Card-based design
- Layout structured using **CSS Float**
- Organized sections for product details and reviews
